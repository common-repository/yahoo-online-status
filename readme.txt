== Description ==

Show YM Online status of all of your members on your widget to show whether they are currently online with Yahoo Messenger or not.

Officially, this online checker feature was already created by Yahoo itself. This plugin just help you to place multiple YM Online Status of your members into one single widget.

Hope you like it :-)

== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
